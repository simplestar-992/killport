# KILLPORT

**Kill Process by Port**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/killport.git
cd killport
go build -o killport .
```

## Usage

```bash
./killport --help
```
