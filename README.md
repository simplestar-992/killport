## Killport

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/killport.git
cd killport
go build -o killport .
```

### Usage

```bash
./killport --help
```

---

## Quick Start

```bash
./killport [command]
```
